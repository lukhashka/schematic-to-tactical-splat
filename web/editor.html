🚪 Двері:

Клікніть на бірюзову лінію (спільна стіна між кімнатами) або на власну зовнішню стіну кімнати, щоб додати двері. Клік по існуючих дверях — видаляє їх.

Ширина дверей фіксована: 1.0 м. Двері на спільній стіні автоматично вирізають прохід в обох кімнатах одночасно, навіть якщо кімнати різного розміру.
Оберіть кімнату для конфігурації або натисніть на [+ НАПРЯМОК] для розширення простору
